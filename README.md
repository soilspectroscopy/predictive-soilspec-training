# Training material on predictive soil spectroscopy

This is the code repository for a training guide on predictive soil
spectroscopy. Originally developed for an in-person workshop in 2023,
this material has been used in multiple training events since then and
is openly maintained for reuse by anyone.

You can access the training material online at:  
<https://soilspectroscopy.github.io/predictive-soilspec-training/>

This guide is organized by the **Soil Spectroscopy for Global Good**
(SS4GG) initiative. SS4GG was originally funded by the [USDA National
Institute of Food and Agriculture award
\#2020-67021-32467](https://cris.nifa.usda.gov/cgi-bin/starfinder/0?path=fastlink1.txt&id=anon&pass=&search=R=89483&format=WEBFMT6NT)
and is currently maintained with institutional support.

To learn more about the SS4GG initiative please visit
<https://soilspectroscopy.org>.
